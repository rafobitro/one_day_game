# Less_Than_One_Day_Game

I just wanted to check how much I can do by spending just one day on one game. Unfortunately, on the day I allocated for this, I couldn't spend the whole day on the project, so I only spent a few hours—making "less than one day game" a better name (or maybe more, because I started creating it more than 24 hours earlier! :)). 

This is a project where I used Raylib to create a simple Flappy Bird-style game with a dynamically changeable background and menu. On another day, I will do a proper one-day game.
